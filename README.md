# Transformer-Related Low-Light Image and Video Enhancement: Companion Repository

This repository is a lightweight academic companion for the survey **"Transformer-Related Low-Light Image and Video Enhancement: A Survey from Restoration Models to Nighttime Vision Applications"**.

## Scope

This repository summarizes representative methods, related restoration backbones, benchmark datasets, evaluation metrics, and resource links for Transformer-related low-light image enhancement (LLIE) and low-light video enhancement (LLVE).

## Taxonomy overview

- **LLIE taxonomy**: End-to-end methods; Decomposition-based methods; Condition-based methods; Frequency-domain methods; Generative methods.
- **LLVE taxonomy**: Alignment-aware spatio-temporal attention; Implicit spatio-temporal attention; Event-guided cross-modal spatio-temporal attention.
