# LLIE taxonomy notes

This taxonomy follows the seed list and groups methods by their primary enhancement strategy.

- **End-to-end methods**: These methods directly map low-light input images to enhanced outputs with unified optimization. See [LLIE methods](../papers/image_methods.md).
- **Decomposition-based methods**: These methods decompose illumination and content-related components, then apply enhancement with decomposition-aware modeling. See [LLIE methods](../papers/image_methods.md).
- **Condition-based methods**: These methods use priors or conditions such as SNR, structure, or illumination to guide adaptive enhancement. See [LLIE methods](../papers/image_methods.md).
- **Frequency-domain methods**: These methods model low-light degradation with frequency-aware or Fourier-related feature processing. See [LLIE methods](../papers/image_methods.md).
- **Generative methods**: These methods use generative modeling, such as diffusion or flow-based strategies, for low-light image enhancement. See [LLIE methods](../papers/image_methods.md).
