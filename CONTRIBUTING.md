# Contributing guidelines

Thank you for contributing.

## Paper updates

- Update paper metadata, paper links, and code links directly in the corresponding paper tables:
  - `papers/image_methods.md`
  - `papers/video_methods.md`
  - `papers/general_restoration_backbones.md`
  - `papers/timeline.md` (when timeline updates are needed)
- Verify updates using reliable sources and keep uncertain fields clearly marked.

## Dataset updates

- Update dataset summaries in `benchmarks/datasets.md` when needed.
- Update dataset access links in `resources/dataset_links.md`.

## Content restrictions

Do not add:

- copyrighted PDF copies
- copied figures
- dataset files
- pretrained weights
- local path references
- unrelated scripts or generated website artifacts

## Writing style

- Keep writing concise, formal, and neutral.
- Avoid unsupported claims.
