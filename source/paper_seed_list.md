# Paper Seed List

This file provides the seed paper list for the companion repository of the survey:

**Transformer-Related Low-Light Image and Video Enhancement: A Survey from Restoration Models to Nighttime Vision Applications**

The list follows the taxonomy used in the survey.  
Please verify paper links, official code links, venue information, and repository status before public release.

Code type should be one of:

- Official
- Unofficial
- Not released
- Project page
- To be checked

Paper URL priority:

1. Publisher page
2. arXiv
3. OpenReview
4. Project page

Code URL priority:

1. Official GitHub
2. Official Gitee
3. Official project page
4. Unofficial implementation, clearly marked
