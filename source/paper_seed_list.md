# Paper Seed List

This file provides the seed paper list for the companion repository of the survey:

**Transformer-Related Low-Light Image and Video Enhancement: A Survey from Restoration Models to Nighttime Vision Applications**

This seed list follows the taxonomy used in the survey. It should be used as the only source of truth when generating the repository Markdown files.

Please verify paper links, official code links, venue information and repository status before public release.

## Writing Standards

All generated Markdown files should use clear, formal and academically appropriate English.

The writing style should be:

- concise but informative
- neutral and non-promotional
- consistent with an academic survey
- free of exaggerated claims
- easy to read for researchers and graduate students

Avoid expressions such as:

- best
- state-of-the-art
- most comprehensive
- complete list
- significantly outperforms
- greatly improves
- revolutionary
- powerful method

Use cautious expressions such as:

- is related to
- is useful for
- provides a reference for
- focuses on
- aims to
- can be used for
- is commonly evaluated on
- remains to be checked

Do not write long subjective comments.  
Each note or key idea should be one concise sentence.

## Link and Code Status Rules

Paper URL priority:

1. Publisher page
2. arXiv
3. OpenReview
4. Project page

Code URL priority:

1. Official GitHub
2. Official Gitee
3. Official project page
4. Unofficial implementation, clearly marked

Code type should be one of:

- Official
- Unofficial
- Project page
- Not released
- To be checked

Do not replace `To be checked` with guessed URLs.

# 6. Metric Seed List

This section is used only to summarize evaluation metrics and their purposes.  
Do not create a separate reference list for performance metrics.  
Do not generate a metric bibliography file.  
Do not add citation columns for metrics unless they are already provided by the user.

| Metric | Type | Task | Direction | Main purpose | Notes |
|---|---|---|---|---|---|
| PSNR | Full-reference | LLIE / LLVE | Higher is better | Measures pixel-level fidelity | Sensitive to misalignment and exposure offsets |
| SSIM | Full-reference | LLIE / LLVE | Higher is better | Measures structural similarity | May not fully reflect perceptual quality |
| LPIPS | Perceptual full-reference | LLIE / LLVE | Lower is better | Measures deep feature similarity | Useful for perceptual restoration analysis |
| NIQE | No-reference | LLIE / real LLVE | Lower is better | Measures natural image quality without references | May not always match human preference in enhanced low-light images |
| BRISQUE | No-reference | LLIE / real LLVE | Lower is better | Measures natural scene statistics without references | Useful when aligned references are unavailable |
| LOE | Luminance consistency | LLIE | Lower is better | Measures lightness order error | Often used to analyze illumination preservation |
| CPCQI | Color and perceptual quality | LLIE | Higher is better | Measures color and perceptual consistency | Useful for color distortion analysis |
| T-Warp-MSE | Temporal consistency | LLVE | Lower is better | Measures warping-based temporal error | Depends on optical flow quality |
| tLPIPS | Temporal perceptual consistency | LLVE | Lower is better | Measures perceptual variation across frames | Useful for flickering analysis |
| ALV | Temporal brightness consistency | LLVE | Lower is better | Measures brightness variation consistency | Implementation details should be reported |
| ALV-Norm | Temporal brightness consistency | LLVE | Lower is better | Measures normalized brightness variation consistency | Should specify luminance channel, cropping and robust averaging settings |
